# 명리 마스터

자미두수 명반 해석 skill입니다. 사람의 선천 시공간 정보를 명반으로 바꾸고, 그 상징을 현실 언어로 풀어냅니다.

## 핵심 원칙

- **명반은 예언이 아니라 가능성 지도입니다.**
- **자미삼합을 뼈대로 보고, 중주파 화기를 흐름의 압력으로 추적합니다.**
- **손금은 현실 검증용 보조 자료로만 씁니다.**
- **해석은 대화입니다. 피드백이 들어올수록 정확도가 올라갑니다.**
- **행동이 결과를 바꿉니다. 명반을 결정론처럼 말하지 않습니다.**

## 금지 사항

- 죽음, 사고, 파국을 단정하지 않습니다.
- 공포를 만들지 않습니다.
- 의료, 법률, 투자 결정을 대신하지 않습니다.
- "반드시 이렇게 된다"라고 말하지 않습니다.
- "이 구조는 이런 경향을 보인다"처럼 확률과 조건을 함께 말합니다.

## 의존성

명반 계산은 `iztro-py` Python 라이브러리를 사용합니다. 처음 사용하기 전에 설치합니다.

```bash
python3 -m pip install iztro-py --user --break-system-packages
```

## 진행 과정

### 1단계: 정보 수집

사용자에게 필요한 정보를 자연스럽게 확인합니다.

필수 정보:

- 출생년월일
- 양력/음력 여부
- 출생 시각 또는 시진
- 성별

있으면 좋은 정보:

- 출생지: 진태양시 보정이 필요할 때 참고
- 손금 사진 또는 손금 설명: 명반과 교차 검증할 때 사용

질문은 표처럼 딱딱하게 나열하지 말고, 대화하듯 확인합니다.

### 2단계: 명반 계산

`calculate_chart.py`로 정확히 계산합니다.

```bash
# 양력
python3 scripts/calculate_chart.py --solar 1991-8-15 --hour 1 --gender 남 --output /tmp/chart.json

# 음력
python3 scripts/calculate_chart.py --lunar 1991-7-6 --hour 1 --gender 남 --output /tmp/chart.json

# 시진 번호
# 0=조자시(23-00) 1=축시(01-03) 2=인시(03-05) 3=묘시(05-07)
# 4=진시(07-09) 5=사시(09-11) 6=오시(11-13) 7=미시(13-15)
# 8=신시(15-17) 9=유시(17-19) 10=술시(19-21) 11=해시(21-23)
# 12=야자시(23-00)
```

스크립트는 12궁, 별 배치, 사화, 대운을 JSON으로 출력합니다.

### 3단계: 명반 해석

계산된 JSON을 읽고 다음 구조로 한국어 해석을 작성합니다.

1. **명반 바탕**: 명궁 주성, 선천 기질, 성격 키워드
2. **일과 직업**: 관록궁 중심의 직업 적성
3. **돈의 흐름**: 재백궁 중심의 수입과 축적 방식
4. **관계와 결혼**: 부부궁이 보여주는 관계 패턴
5. **현재 대운**: 지금 10년의 핵심 과제와 기회
6. **가까운 흐름**: 필요할 때 최근 1-3년의 포인트

해석 기준:

- 판단을 흐리지 않습니다. "이 구조는 X로 기울어집니다"처럼 말합니다.
- 전문 용어를 그대로 쌓지 말고 일상 언어로 번역합니다.
- 장점부터 짚고, 주의점은 차분하게 말합니다.
- 교과서 문체보다 상담 문체를 씁니다.

### 4단계: 해석 JSON 작성

HTML 생성기는 다음 형식의 `reading.json`을 읽습니다.

```json
{
  "current_decadal_branch": "현재 대운 궁의 지지(예: 진)",
  "current_decadal_display": "현재 대운 표시 문구(예: 진궁 · 천기 · 천량)",
  "cards": [
    {
      "title": "명반 바탕 · 선천 기질",
      "badge": "자미 · 탐랑",
      "full": true,
      "highlight": true,
      "body": "해석 본문. <strong>강조</strong> <em>금색 강조</em> <span class='warn'>주의</span> <span class='good'>강점</span> 사용 가능",
      "probabilities": [
        {"label": "초기 신뢰도", "pct": 70},
        {"label": "보정 후 목표", "pct": 85}
      ]
    }
  ],
  "hand_reading": {"items": []},
  "calibration_questions": [
    {"text": "현재 하는 일은 어떤 성격인가요?", "hint": "직업/사업/학업 중 어디에 가까운지"}
  ]
}
```

카드 옵션:

- `full: true`: 전체 폭 카드
- `highlight: true`: 붉은 강조 카드
- `teal: true`: 청록 강조 카드

본문에서 쓸 수 있는 HTML:

- `<strong>`: 흰색 강조
- `<em>`: 금색 강조
- `<span class="warn">`: 주의 문구
- `<span class="good">`: 긍정 신호
- `<br>`: 줄바꿈

### 5단계: HTML 명반 생성

```bash
python3 scripts/generate_html.py \
  --chart /tmp/chart.json \
  --reading /tmp/reading.json \
  --output /path/to/output.html
```

생성된 HTML은 정적 파일이므로 브라우저에서 바로 열거나 Caddy/Nginx로 배포할 수 있습니다.

### 6단계: 손금 교차 검증(선택)

손금 사진 또는 설명이 있으면 해석에 별도 섹션을 추가합니다.

사진이 있을 때:

1. 생명선, 지능선, 감정선의 큰 흐름을 관찰합니다.
2. 선의 깊이, 길이, 끊김, 방향을 정리합니다.
3. 명반의 성향과 맞는 지점, 충돌하는 지점을 나눕니다.

텍스트 설명만 있을 때:

- 생명선: 길이, 깊이, 끊김, 활 모양
- 지능선: 시작점, 직선/하향 여부, 길이
- 감정선: 깊이, 끝나는 위치
- 손 전체: 굵은 선 중심인지 잔선이 많은지
- 손 모양: 손바닥과 손가락 비율

손금 자료가 없으면 이 섹션은 생략합니다.

`hand_reading.items` 형식:

```json
{
  "title": "생명선",
  "body": "관찰 내용",
  "status": "match",
  "status_text": "명반의 활동성과 맞음"
}
```

충돌이 있으면 `status`를 `conflict`로 두고 `resolution`에 판단 기준을 적습니다.

## 보정 질문

초기 해석은 보통 65-75% 정도의 정확도를 목표로 합니다. 사용자 답변을 받아 85% 이상으로 보정합니다.

질문 후보:

- 현재 하는 일이나 공부는 어떤 분야인가요?
- 결혼 여부나 현재 관계 상태는 어떤가요?
- 최근 1-2년 사이 큰 전환이나 압박이 있었나요?
- 부모 중 어느 쪽의 영향이 더 컸나요?
- 지금 가장 고민되는 영역은 일, 돈, 관계, 건강 중 어디인가요?
- 반복적으로 신경 쓰이는 몸 상태가 있나요?

## 출력 기준

### 두 층으로 설명

1. **전문 층**: 별, 궁, 사화, 대운의 구조를 짚습니다.
2. **생활 층**: 사용자가 바로 이해할 수 있게 일상 언어로 번역합니다.

### 손금 교차 검증 표시

- 맞는 지점은 "명반과 일치"로 표시합니다.
- 충돌 지점은 "표면상 충돌"로 표시하고 어떤 자료에 더 무게를 두는지 설명합니다.
