# 사화 참고

사화는 자미두수에서 흐름을 움직이는 핵심 장치입니다. 천간마다 네 별이 화록, 화권, 화과, 화기로 변합니다.

## 사화의 의미

| 사화 | 핵심 의미 | 비유 | 방향 |
| --- | --- | --- | --- |
| **화록** | 재물, 순조로움, 누림 | 자원이 자연스럽게 들어오는 문 | 대체로 긍정, 자원 증가 |
| **화권** | 권한, 통제, 주도권 | 마이크를 잡고 판을 움직이는 힘 | 중립에서 긍정, 통제력 증가 |
| **화과** | 명예, 학습, 인정 | 공개적으로 이름이 불리는 장면 | 긍정, 평판과 귀인 증가 |
| **화기** | 집착, 막힘, 소모 | 계속 마음에 걸리는 매듭 | 부담이지만 중요한 관심 지점 |

| 천간 | 화록 | 화권 | 화과 | 화기 |
| --- | --- | --- | --- | --- |
| 갑 | 염정 | 파군 | 무곡 | 태양 |
| 을 | 천기 | 천량 | 자미 | 태음 |
| 병 | 천동 | 천기 | 문창 | 염정 |
| 정 | 태음 | 천동 | 천기 | 거문 |
| 무 | 탐랑 | 태음 | 우필 | 천기 |
| 기 | 무곡 | 탐랑 | 천량 | 문곡 |
| 경 | 태양 | 무곡 | 태음 | 천동 |
| 신 | 거문 | 태양 | 문곡 | 문창 |
| 임 | 천량 | 자미 | 좌보 | 무곡 |
| 계 | 파군 | 거문 | 태음 | 탐랑 |

`iztro-py` 계산 결과의 사화 데이터를 우선 사용합니다. 이 표는 해석을 확인하는 보조 자료입니다.

## 궁별 해석 방향

### 화록

| 궁 | 해석 방향 |
| --- | --- |
| 명궁 | 선천적으로 일이 풀릴 여지가 있고 사람을 끌어당기는 힘이 있습니다. |
| 재백궁 | 돈을 만드는 감각이 좋고 자원이 모이기 쉽습니다. |
| 관록궁 | 일에서 기회와 도움을 받기 쉽습니다. |
| 부부궁 | 관계에서 따뜻함과 호의가 들어오기 쉽습니다. |
| 천이궁 | 밖으로 나가면 기회와 사람이 열립니다. |

### 화권

| 궁 | 해석 방향 |
| --- | --- |
| 명궁 | 스스로 주도권을 잡으려는 힘이 강합니다. |
| 관록궁 | 일에서 결정권과 책임이 커집니다. |
| 재백궁 | 돈을 통제하고 지키려는 감각이 좋습니다. |
| 부부궁 | 관계에서 주도권 문제가 중요해집니다. |

### 화과

| 궁 | 해석 방향 |
| --- | --- |
| 명궁 | 공부, 평판, 인정과 연결됩니다. |
| 관록궁 | 전문성이나 이름값으로 일이 풀릴 수 있습니다. |
| 부모궁 | 부모, 윗사람, 교육의 영향이 좋게 작용합니다. |
| 복덕궁 | 정신세계, 취향, 문화적 감각이 풍부합니다. |

### 화기

화기는 가장 먼저 봐야 할 지점입니다. 단순한 불운이 아니라, 마음이 오래 붙잡히고 에너지가 새는 곳입니다.

| 궁 | 해석 방향 |
| --- | --- |
| 명궁 | 자기 기준이 높고 자기 부정으로 기울 수 있습니다. |
| 관록궁 | 일에서 압박, 지연, 불만족이 생기기 쉽습니다. |
| 재백궁 | 돈의 누수나 불안이 생기기 쉬워 관리가 필요합니다. |
| 부부궁 | 관계에서 기대와 실망이 반복될 수 있습니다. |
| 복덕궁 | 생각이 많고 마음이 쉬지 못하기 쉽습니다. |
| 질액궁 | 몸의 신호를 무시하지 말고 점검해야 합니다. |
| 천이궁 | 이동, 외부 환경, 낯선 사람과의 마찰에 주의합니다. |

## 중주파 화기 추적

중주파 관점에서는 화기의 연결을 중요하게 봅니다.

1. **생년 화기**: 평생 반복되는 선천 집착 지점
2. **대운 화기**: 이 10년의 주요 압박과 과제
3. **유년 화기**: 해당 해의 구체적 사건 압력

생년 화기, 대운 화기, 유년 화기가 같은 궁이나 같은 주제로 겹치면 그 영역은 과장되어 드러납니다.

## 해석 순서

1. 먼저 화기를 봅니다. 이야기가 생기는 곳입니다.
2. 다음으로 화록을 봅니다. 소모를 풀 자원입니다.
3. 화권과 화과는 주도권과 평판의 보조 축으로 봅니다.
4. 사화가 겹치면 영향이 커진다고 판단합니다.
